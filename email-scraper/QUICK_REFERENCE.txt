💾 DATABASE
═══════════════════════════════════════════════════════════════════════════════

Tables:
  emails   - All extracted emails (dedup via PRIMARY KEY)
  domains  - Domain aggregation
  jobs     - Job history

Query examples:
  sqlite3 scraped_data.db "SELECT * FROM emails LIMIT 10;"
  sqlite3 scraped_data.db "SELECT domain, email_count FROM domains ORDER BY email_count DESC LIMIT 20;"
  sqlite3 scraped_data.db "SELECT * FROM jobs;"

📋 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════
"Connection refused"
  → Run: python server.py

"Cannot find module fastapi"
  → Run: pip install -r requirements.txt

"npm not found"
  → Install Node.js from nodejs.org

"Frontend can't reach API"
  → Check backend is running on port 8000
  → Check browser console (F12) for errors

"Slow queries on SQLite"
  → Upgrade to PostgreSQL for millions+ records
  → See ROADMAP.md for production setup

More help: See QUICKSTART.md section "Troubleshooting"
