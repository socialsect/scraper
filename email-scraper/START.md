# 🚀 Quick Start — Simple Flask HTML Frontend

## Setup (One-Time)

```bash
pip install -r requirements.txt
```

## Run (3 Easy Commands)

**Terminal 1 — FastAPI Backend:**
```bash
python server.py
```
✅ API on http://localhost:8000

**Terminal 2 — Flask HTML Frontend:**
```bash
python app.py
```
✅ UI on http://localhost:5000

**Browser:**
Open http://localhost:5000

---

## That's It!

- Fill in query
- Click "Start Job"
- Watch live updates (polls every 1 second)
- Download CSV when done

No npm, no React, no SSE complexity. Just works. 🎯
